# 練習サイト

## 制作背景
本を通してコーディング力を体系的に向上させることを目的に制作しました。
特に、実務で求められる「レイアウト設計力」と「再現性のあるマークアップ力」を身につけるため、レスポンシブレイアウトの基礎となる可変レイアウト・メディアクエリ・単位の使い分け・主要なレイアウト手法の仕組みについて重点的に学習しました。
また、単なる静的なページ作成ではなく、実務レベルのコンポーネントを再現することを通して、実践的なコーディングスキルの習得を目指しました。

---

## 使用技術
HTML / CSS / JavaScript

---

## 工夫した点
実務でよく使用されるUIコンポーネントを意識して設計・実装しました。 具体的には、アスペクト比の制御、背景色エリアの設計、カードレイアウト、ブロークングリッドレイアウトなどを取り入れ、見た目だけでなく構造的にも再現性の高い実装を心がけました。 また、用途や構造に制約のあるフォームにおいては、HTMLの文法的正しさを維持しつつ、アクセシビリティやユーザビリティにも配慮したマークアップを行いました。CSS設計においても、再利用性と保守性を意識してクラス設計を行っています。

---

## 苦労した点
レスポンシブ対応において、単に見た目を合わせるだけでなく、レイアウトの意図を崩さずに画面幅ごとの挙動を整理する点に苦労しました。
特に、カードレイアウトやグリッド系のデザインでは、崩れ方や余白の取り方によって印象が大きく変わるため、細かな調整が必要でした。
また、HTMLの意味構造を意識したマークアップと、見た目重視の設計とのバランスを取ることにも時間がかかりました。

---

## 学んだこと
HTMLは単なる表示のための記述ではなく、意味を持った構造として設計することで、アクセシビリティや将来的な保守性が大きく変わることを実感しました。
CSS設計についても、場当たり的な記述ではなく、再利用性や拡張性を意識することで、コード全体の品質が大きく向上することを理解しました。
